# Cognitive Modeling Knowledge Base Glossary

This comprehensive glossary provides definitions and cross-references for key terms used throughout the cognitive modeling knowledge base. Terms are organized alphabetically and include domain-specific contexts where relevant.

## A

### Active Inference
**Definition**: A theoretical framework proposing that all cognitive processes can be understood as inference on generative models of the world, where action is inference about preferred outcomes.

**Domains**: Cognitive Science, Mathematics, Systems Theory
**Related Terms**: [[mathematics/free_energy_principle|Free Energy Principle]], [[cognitive/generative_model|Generative Model]], [[systems/emergence|Emergence]]

### Adaptive Systems
**Definition**: Systems that can modify their behavior or structure in response to environmental changes to improve performance.

**Domains**: Systems Theory, Biology, Engineering
**Related Terms**: [[systems/self_organization|Self-Organization]], [[biology/evolutionary_adaptation|Evolutionary Adaptation]], [[systems/feedback_loops|Feedback Loops]]

### Agent-Based Modeling
**Definition**: Computational method for simulating complex systems by representing individual agents and their interactions.

**Domains**: Systems Theory, Computer Science, Biology
**Related Terms**: [[systems/multi_agent_systems|Multi-Agent Systems]], [[biology/swarm_intelligence|Swarm Intelligence]], [[mathematics/complex_systems|Complex Systems]]

### Allostasis
**Definition**: The process by which the body achieves stability through change, anticipating and adapting to future demands.

**Domains**: Biology, Physiology, Psychology
**Related Terms**: [[biology/homeostasis|Homeostasis]], [[cognitive/stress_response|Stress Response]], [[systems/adaptive_capacity|Adaptive Capacity]]

## B

### Bayesian Inference
**Definition**: Statistical method for updating beliefs about unknown quantities based on observed data using Bayes' theorem.

**Domains**: Mathematics, Statistics, Cognitive Science
**Related Terms**: [[mathematics/bayes_theorem|Bayes' Theorem]], [[cognitive/probabilistic_inference|Probabilistic Inference]], [[mathematics/posterior_distribution|Posterior Distribution]]

### Belief Propagation
**Definition**: Algorithm for computing marginal probabilities in graphical models through iterative message passing.

**Domains**: Mathematics, Computer Science, Cognitive Science
**Related Terms**: [[mathematics/graphical_models|Graphical Models]], [[cognitive/message_passing|Message Passing]], [[mathematics/factor_graphs|Factor Graphs]]

### Biological Network
**Definition**: Complex interconnected system of biological entities (genes, proteins, cells) that interact to perform functions.

**Domains**: Biology, Systems Biology, Mathematics
**Related Terms**: [[biology/gene_regulatory_networks|Gene Regulatory Networks]], [[biology/metabolic_networks|Metabolic Networks]], [[mathematics/network_theory|Network Theory]]

### Bounded Rationality
**Definition**: Theory that decision-making is limited by cognitive constraints, information availability, and time.

**Domains**: Psychology, Economics, Cognitive Science
**Related Terms**: [[cognitive/decision_making|Decision Making]], [[cognitive/cognitive_biases|Cognitive Biases]], [[mathematics/optimization_theory|Optimization Theory]]

**Related Terms**: [[cognitive/swarm_intelligence|Swarm Intelligence]], [[systems/emergence|Emergence]], [[biology/social_behavior|Social Behavior]]

### Complex Adaptive System

**Domains**: Systems Theory, Biology, Social Science
**Related Terms**: [[systems/complex_systems|Complex Systems]], [[systems/adaptive_systems|Adaptive Systems]], [[biology/evolutionary_dynamics|Evolutionary Dynamics]]

**Related Terms**: [[mathematics/limit_theory|Limit Theory]], [[cognitive/learning_convergence|Learning Convergence]], [[systems/system_stability|System Stability]]
